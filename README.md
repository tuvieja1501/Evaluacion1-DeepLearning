## Descripción
Este repositorio contiene la Evaluación Parcial N° 1. Se implementa una Red Neuronal Artificial (MLP) utilizando TensorFlow/Keras para predecir la fuga de clientes en una empresa de telecomunicaciones.

## Instrucciones para ejecutar el proyecto
1. Abre el archivo `Evaluacion1_DeepLearning.ipynb` alojado en este repositorio.
2. Haz clic en el botón azul superior que dice **"Open in Colab"** (Abrir en Colab).
3. Una vez en Google Colab, ve al menú superior y selecciona **Entorno de ejecución** > **Ejecutar todas**.
4. Las librerías necesarias (`pandas`, `scikit-learn`, `tensorflow`) ya están preinstaladas en el entorno de Colab, por lo que no se requiere instalación local adicional.
